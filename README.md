## How to use?

The application works with commands, there are commands to change views or handlers and commands for each of them:

### Change view

```
view books
view patrons
view loans
view reports
view fines
```

Commands for each view

### books:

```
new
delete
update
show all
show by genre
find by title
find by author
find by ISBN
```

### patrons:

```
new
delete
update
show all
find by name
find by m-number
```

### loans:

```
lend
return

```

### reports:

```
show overdue books
show borrowed books
show current loans by patron
show loans by patron
show most borrowed books
show most active patrons
show patron debts
```

### fines:

```
pay
make
show all
show actives
```
